┌─ KEY CALCULATIONS ────────────────────────────────────────────────────────────┐
│                                                                                 │
│  TOTAL DUE (automatic when creating loan):                                     │
│  ┌─────────────────────────────────────────────────────────────┐               │
│  │ total_due = principal + (principal × interest_rate / 100)   │               │
│  │ Example: 10,000 + (10,000 × 10 / 100) = 11,000             │               │
│  └─────────────────────────────────────────────────────────────┘               │
│                                                                                 │
│  BALANCE (updated automatically when payment recorded):                        │
│  ┌─────────────────────────────────────────────────────────────┐               │
│  │ balance = total_due - SUM(all payments for this loan)       │               │
│  │ When balance ≤ 0: status = 'paid' (automatic)               │               │
│  └─────────────────────────────────────────────────────────────┘               │
│                                                                                 │
│  DAYS PAST DUE (for reporting):                                                │
│  ┌─────────────────────────────────────────────────────────────┐               │
│  │ days_overdue = TODAY - first_due_date                       │               │
│  │                                                              │               │
│  │ Buckets:                                                     │               │
│  │ ├─ Current: days_overdue ≤ 0 (not yet due)                 │               │
│  │ ├─ 1-30: days_overdue between 1 and 30                      │               │
│  │ ├─ 31-60: days_overdue between 31 and 60                    │               │
│  │ ├─ 61-90: days_overdue between 61 and 90                    │               │
│  │ └─ 90+: days_overdue > 90                                   │               │
│  └─────────────────────────────────────────────────────────────┘               │
│                                                                                 │
└─────────────────────────────────────────────────────────────────────────────────┘
